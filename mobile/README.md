# LifeVault AI — Mobile App (Expo)

This is the mobile companion app for LifeVault AI, built with React Native and Expo.

## Features
- **Real-time Alerts**: Push notifications for confirmed threats.
- **Biometric Vaults**: Unlock your security containers on the go.
- **Scam Detector**: Mobile-optimized URL and text analysis.

## Development Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npx expo start
   ```
3. Open on your device via the Expo Go app.

## Security
- This app uses `lucide-react-native` for high-fidelity security icons.
- Designed to integrate with the LifeVault Backend API.
